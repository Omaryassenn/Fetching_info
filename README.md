# Arabic Player Data Search App

# Description
The Arabic Player Data Search App is a React application designed to help users search for player information using an intuitive search interface. The application allows users to enter search queries and retrieve player details from a JSON dataset containing Arabic data.

The app offers the following features:

User-friendly search bar for entering player names or other search criteria.
Real-time filtering of player data based on search queries.
Display of search results in a tabular format, organized for easy readability.
Support for complex JSON data structures, including nested objects.
Convenient display of player information, including name, birthdate, nationality, contact details, and more.
This project showcases the use of React state management, JSON data manipulation, and dynamic rendering to create a seamless and efficient search experience. It's an ideal example of building a real-world data search and visualization tool using modern web development technologies.

# Getting Started
## Run Locally

Clone the project

```bash
  git clone https://github.com/adhamali450/broskies-yearbook
```

Go to the project directory

```bash
  cd broskies-yearbook
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
# Data Source
The player data used in this app is sourced from the playerDataArabic.json file, which contains information about various players, including their names, birthdates, nationalities, and more. The app dynamically searches and filters this dataset based on user input.
